# color

This package adds color verbs to fmt.Printf

Foreground Colors:
%h#fgBlack#
%h#fgRed#
%h#fgGreen#
%h#fgBlue#
%h#fgMagenta#
%h#fgCyan#
%h#fgWhite#
%h#fgBrighBlack#
%h#fgBrightRed#
%h#fgBrightGreen#
%h#fgBrightBlue#
%h#fgBrightMagenta#
%h#fgBrightCyan#
%h#fgBrightWhite#

Background Colors:
%h#bgBlack#
%h#bgRed#
%h#bgGreen#
%h#bgBlue#
%h#bgMagenta#
%h#bgCyan#
%h#bgWhite#
%h#bgBrighBlack#
%h#bgBrightRed#
%h#bgBrightGreen#
%h#bgBrightBlue#
%h#bgBrightMagenta#
%h#bgBrightCyan#
%h#bgBrightWhite#

256 Colors:
%h#144#

For true color:
%h#rgb(144,155,166)#

Attributes:
%h#reset#
%h#bold#
%h#faint#
%h#italic#
%h#underline#
%h#blink#
%h#inverse#
%h#invisible#
%h#crossedOut#

To combine:
%h#fgBlue+bgRed+bold#
